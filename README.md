# Expert Maker Overlay

This is the standalone overlay/editor.

Open `index.html` directly for the editor, or run `Expert Maker Studio.cmd` for the local export server and MP4 rendering.

Controls:

- `Pause` / `Edit` manage the overlay playback and editing mode.
- `Load screenshot` places any image behind the overlay for local editing.
- `Page URL` loads a web interface into the target frame when the page allows iframe embedding.
- `Clear target` returns to the neutral background.
- Press `Z` to hide/show the control panel.

The overlay state is stored in browser localStorage under `expert_maker_overlay_layout_v1`.
